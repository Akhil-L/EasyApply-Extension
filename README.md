# EasyApply Extension

AI-powered Chrome extension that autofills job applications.

## Features
- Universal autofill
- Resume upload
- LinkedIn/Naukri/Indeed support
- Dynamic React forms support
- Smart field detection

## Tech Stack
- React
- TypeScript
- Vite
- Chrome Extension Manifest V3

## Installation
pnpm install
pnpm build

Load unpacked from dist/
